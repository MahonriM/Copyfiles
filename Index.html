Copy files with robocopy

  • Introduction
  • Description

In this website you will learn about the use of robocopy command for copy files in windows. This command help us to copy and move files and folders from one place to other unfortunately this commad only works for windows system.


We can use this command to copy and move files, folders to a specific destination


Instructions

  • Open command prompt with admin rights.

  • Write the next command robocopy \drivename \folder \otherfolder \ drivename /E /mt /z and press enter
  • The first part of robocopy is the source and the second part is the destination

  • /E Copy subfolders including empty subfolders /mt allows you to specify the number of threads robocopy will use to copy files. The maximum is 128. /z Copy files in restartable mode

  • Sometimes some files are missing and it fails to copy however most of the times It will help you to move and copy files

Examples

For move files from one location to another one in the same PC or drive


  • robocopy C:\Users\Music\musica1 C:\Users\Videos /E /mt /z /xo

  • robocopy F:\Music D:\Music /E /mt /z /xo
